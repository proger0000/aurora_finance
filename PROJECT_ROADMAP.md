# Дорожная карта проекта "Aura Finance"

Этот документ описывает ключевую идею, уже реализованный функционал и план дальнейшей разработки для отслеживания прогресса.

## 1. Ключевая идея проекта

**Aura Finance** — это прогрессивное веб-приложение (PWA) нового поколения для управления личными и семейными финансами. Проект сочетает интуитивный дизайн с мощной аналитикой, уникальным модулем для отслеживания автомобилей и интегрированным AI-ассистентом "Aura".

**Цель:** Стать персональным другом-ментором в мире финансов.
**Философия:** "Простота снаружи, мощность внутри".

---

## 2. Статус выполнения работ

### ✅ Реализованный функционал (Прототип)

На данный момент заложена прочная основа и реализован пользовательский интерфейс для большинства ключевых разделов.

- **[x] Архитектура фронтенда:**
    -   Проект настроен с использованием Vite, React 19 и TypeScript.
    -   Создана логическая структура компонентов.
    -   Настроена работа с переводами (i18n) для английского и украинского языков.
- **[x] UI-кит и дизайн:**
    -   Создан компонент `<Card>` с эффектом "матового стекла" (frosted glass), который является основой дизайна.
    -   Реализован многократно используемый компонент модального окна `<Modal>`.
    -   Создан компонент `<ProgressBar>` для визуализации прогресса.
- **[x] Ключевые разделы (UI и мок-логика):**
    -   **`Dashboard`**: Главный экран с общей сводкой (баланс, доходы/расходы, диаграмма трат, цели).
    -   **`Transactions`**: Экран со списком транзакций и формой для добавления доходов/расходов через модальное окно.
    -   **`Goals`**: Раздел для управления целями накоплений. Реализовано добавление и удаление целей.
    -   **`Reports`**: Базовая версия отчетов с графиком "Доходы vs. Расходы" от `recharts`.
    -   **`Garage`**: Начальная реализация раздела для отслеживания авто с базовыми расчетами и графиками.
    -   **`AIAssistant`**: Функциональный чат-интерфейс с интеграцией `geminiService` для ответов на запросы пользователя.
    -   **`Settings`**: Полностью рабочий раздел для смены темы (светлая/темная), языка и основной валюты.
- **[x] Система данных (Mock API):**
    -   Создан сервис `apiService.ts`, симулирующий работу бэкенда с задержками.
    -   Реализован хук `useData`, который централизованно получает все данные для приложения.
